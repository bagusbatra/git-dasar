## 📌 Deskripsi

Repository ini berisi materi **Git Dasar** untuk pemula yang ingin memahami cara kerja Git dan GitHub dari nol.

Materi disusun secara **bertahap + praktik langsung**, sehingga cocok untuk:

* 🧑‍🎓 Siswa pemula
* 👨‍🏫 Pembelajaran di kelas
* 💻 Belajar mandiri

---

## 🎯 Tujuan Pembelajaran

Setelah menyelesaikan materi ini, kamu akan bisa:

* Memahami apa itu Git & GitHub
* Menggunakan perintah dasar Git
* Menyimpan perubahan (commit)
* Mengirim project ke GitHub (push)
* Melakukan **Pull Request (PR)**

## 🚀 Cara Menggunakan Repository

### 1. Fork Repository Ini

Klik tombol **Fork** di pojok kanan atas.

---

### 2. Clone Repository

```bash
git clone https://github.com/USERNAME/git-dasar-untuk-pemula.git
cd git-dasar-untuk-pemula
```

---

### 3. Pelajari Materi

Buka folder:

```
materi/
```

Pelajari secara berurutan:

1. Apa itu Git
2. Instalasi
3. Perintah dasar

---

### 4. Kerjakan Latihan

Masuk ke folder:

```
praktik/
```

Ikuti instruksi pada file latihan.

---

## 📥 Cara Mengumpulkan Tugas (WAJIB)

### 1. Masuk ke folder:

```
pengumpulan/
```

---

### 2. Buat folder dengan nama kamu

Contoh:

```
pengumpulan/bagus/
```

---

### 3. Masukkan file hasil latihan

Contoh:

```
halo.txt
```

---

### 4. Commit & Push

```bash
git add .
git commit -m "tugas git dasar - [nama]"
git push
```

---

### 5. Buat Pull Request

Klik:
👉 **Compare & Pull Request**

---

### 6. Isi format berikut:

```
Nama :
Materi :
Catatan :
```

---

## ⚠️ Hal Penting yang Harus Diperhatikan

* ❗ WAJIB fork repository terlebih dahulu
* ❗ Jangan push ke repo utama (akan ditolak)
* ❗ Pastikan sudah login GitHub di Git
* ❗ Gunakan nama file yang jelas

---

## 💡 Tips Belajar

* Gunakan `git status` untuk mengecek perubahan
* Gunakan pesan commit yang jelas
* Jangan takut error — itu bagian dari belajar 🔥
