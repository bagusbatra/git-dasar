# 📌 Apa Itu Git?

Git adalah sistem untuk mencatat perubahan file.

Bayangkan seperti:
📖 Buku catatan versi coding

Setiap perubahan yang kamu lakukan akan tersimpan.

---

# 🎯 Kenapa Kita Pakai Git?

✅ Bisa melihat perubahan
✅ Bisa kembali ke versi sebelumnya
✅ Bisa kerja bareng teman
✅ Digunakan oleh programmer di seluruh dunia

---

# 🌐 Apa Itu GitHub?

GitHub adalah tempat menyimpan project Git secara online.

Git = alat
GitHub = tempat penyimpanan

---

# 🔥 Kesimpulan

Git membantu kita:
👉 Mengatur dan menyimpan project dengan rapi
