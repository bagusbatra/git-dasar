# 📌 Apa Itu Git?

Git adalah sebuah **sistem kontrol versi (Version Control System)**
yang digunakan untuk **mencatat setiap perubahan pada file atau project**.

---

## 🧠 Penjelasan Sederhana

Bayangkan kamu sedang mengerjakan tugas:

📄 `tugas_v1.docx`
📄 `tugas_v2.docx`
📄 `tugas_final.docx`
📄 `tugas_final_fix.docx` 😵

👉 Ribet dan membingungkan, kan?

---

## 💡 Dengan Git

Kamu cukup punya **1 file saja**, tapi:

* Semua perubahan akan **tersimpan otomatis**
* Kamu bisa **kembali ke versi sebelumnya kapan saja**
* Tidak perlu membuat banyak file duplikat

---

## 📖 Analogi Git

Git seperti:

> 📖 Buku catatan pintar untuk coding

Setiap kamu mengubah sesuatu, Git akan:

* Menyimpan perubahan
* Memberi riwayat (history)
* Bisa membuka kembali catatan lama

---

# 🎯 Kenapa Kita Pakai Git?

Git digunakan karena sangat membantu dalam pengembangan project:

✅ Melihat perubahan yang terjadi
✅ Kembali ke versi sebelumnya
✅ Menyimpan riwayat pekerjaan
✅ Bekerja sama dengan tim
✅ Digunakan oleh programmer di seluruh dunia

---

# 🔄 Contoh Alur Penggunaan Git

```text
Edit file → Simpan (commit) → Lihat riwayat → Ulangi
```

👉 Setiap “commit” = satu catatan perubahan

---

# 🌐 Apa Itu GitHub?

GitHub adalah platform untuk:
👉 Menyimpan project Git secara **online (cloud)**

---

## 📦 Perbedaan Git dan GitHub

| Git                             | GitHub                               |
| ------------------------------- | ------------------------------------ |
| Alat (software)                 | Tempat penyimpanan online            |
| Jalan di komputer               | Jalan di internet                    |
| Digunakan untuk version control | Digunakan untuk sharing & kolaborasi |

---

## 💡 Analogi

* Git = Buku catatan 📖
* GitHub = Google Drive 📂

---

# 🚀 Kenapa Kita Pakai GitHub?

Dengan GitHub, kamu bisa:

🌍 Menyimpan project secara online
👨‍💻 Berbagi dengan teman
🤝 Kolaborasi dalam satu project
📊 Menunjukkan portofolio coding

---

# 🔥 Kesimpulan

Git membantu kita untuk:
👉 Mengatur perubahan project dengan rapi
👉 Menyimpan riwayat pekerjaan
👉 Bekerja lebih profesional seperti programmer

---

## 💬 Catatan Penting

> tapi tentang cara berpikir dalam mengelola project.”
