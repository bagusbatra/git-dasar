# ⚙️ Install Git

## 💻 Windows

1. Download Git dari:
   https://git-scm.com

2. Install seperti biasa (klik next terus)

---

## ✅ Cek apakah berhasil

Buka CMD / Terminal:
```
git --version
```
Jika muncul versi, berarti berhasil 🎉

---

# 🔧 Konfigurasi Awal

Masukkan nama dan email:
```
git config --global user.name "Nama Kamu"
git config --global user.email "[email@gmail.com]"
```
---

# 📌 Kenapa ini penting?

Agar setiap commit ada identitasnya
