# 📥 Cara Pengerjaan dan Mengumpulkan Tugas

1. Fork repository ini

2. Masuk ke repo kamu

3. Clone repo hasil fork

    ```
    git clone https://github.com/akun-siswa/git-dasar.git
    cd git-dasar
    ```
4. Kerjakan Tugas (WAJIB SIMULASI)
    * masuk ke folder
    ```
    pengumpulan/
    ``` 
    * buat folder sesuai nama kamu
    ```
    pengumpulan/bagus/
    ```

5. Buat file:

   * halo.txt
   * isi dengan teks berikut :
   ```
   Halo ini latihan git saya
   ```

6. Commit & Push:
    ```
   git add .
   git commit -m "tugas git dasar - [nama]"
   git push
   ```

7. Buat Pull Request (WAJIB)
    * Buka repo fork
    * klik 
    ```
    compare & pull request
    ```

8. Isi deskripsi PR
    * contoh
    ```
    Nama : Bagus
    Materi : Dasar Git
    Catatan : - [isi jika ada]
    ```
